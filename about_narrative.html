About Larry Volz

Hi everyone, I'm Larry. Sometimes people call me "Doc".

I'm a software engineer/web developer with some pretty interesting and fun "side gigs".

As a web developer I have created sites using WordPress and other CDN's, hand-coded HTML/CSS/Bootstrap sites (like this one), and quite a lot of JavaScript/jQuery customization.

As a software engineer I recently built a memorial website-builder/flower shop for the funeral industry. This is built with Python/Flask using RESTful API integration with OAuth and a PostgreSQL database. I integrated jQuery-UI with SQLAlchemy for an autocorrect that communicated directly to PostgreSQL on the back end and launched it on Heroku cloud services platform.

Years ago, I built a full-stack CRM for a national therapist referral company using JavaScript and PHP/SQL on the back end. It was developed iteratively using feedback from our sales team and included API's for geo-locating and sales automation.

I have a bachelors degree in science from Radford University and recently earned a second degree in computer science from University of the People. I have taken college coursework in Java, Python, operating systems, software engineering, networking & communications, SQL and web programming.

Recently I was also accepted into and have been attending Springboard's competitive full-stack software engineering bootcamp where I am working very hard at a great curriculum and have been honing my skills in advanced Javascript and it's libraries, Python tools like Flask, PostgreSQL and SQLAlchemy, security protocols like WTForms, and RESTful API's. Within the next few months I will be able to boast a strong proficiency in NodeJS, Redux and React as well.

I am VERY excited about joining a creative team where we can work together full-time making beautiful and effective code.

As an escape room owner/designer I have coded in C/C++, Python/Flask, JavaScript and Java on Arduino and Raspberry Pi microcontrollers. These were embedded systems I engineered and built for puzzles designed to challenge and entertain my players. (moo ha haaaa!!) These included systems that guided laser mazes, animated props, unlocked secret doors and more.

Creating is one of the things I enjoy most in life. You might have picked up on that from all the crazy pictures of me.

I spent my younger years traveling the world as an illusionist and comedy hypnotist. And then later I have spent a lot of time helping people challenge and change bad habits through hypnotherapy.

So I like to think that I'm fun to be around, a VERY dedicated and self-starting project manager and have great communication and motivational soft skills.
